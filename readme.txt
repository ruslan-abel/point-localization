	Структура классов (примерная, более точная опубликуется в док):

	Point(Точка) 				содержит координаты точки типа double
	Vertices(Вершины) 			вектор Point
	Edge(Ребро) 				пара из индексов точек
	Edges(Рёбра) 				вектор Edge
	Edge_List                   рёберный узел
	Edge_List(РСДС)				рёберный список с двойными связями
    Predicat                    базовый класс сравнения
	PPLG(ППЛГ)                  основная структура
